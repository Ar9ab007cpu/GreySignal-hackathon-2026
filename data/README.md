# GreySignal Data

`real_training_dataset.csv` is generated by:

```powershell
.\.venv\Scripts\python.exe scripts\build_training_dataset.py --start-year 2020 --end-year 2025 --fast
```

The current generated dataset uses only real observed external-data columns:

- World Bank WDI indicators: GDP growth, inflation, unemployment, FDI, population
- World Bank WGI bulk CSV indicators: political stability, government effectiveness, regulatory quality, rule of law, control of corruption
- Optional GDELT event counts
- Optional UN Comtrade import values

The ML target is:

```text
final_risk_score = 100 - World Bank WGI Political Stability governance score
```

Run without `--fast` to also attempt GDELT and UN Comtrade collection:

```powershell
.\.venv\Scripts\python.exe scripts\build_training_dataset.py --start-year 2020 --end-year 2025
```

GDELT and UN Comtrade can be slower or rate-limited, so the fast build is better for local development.
